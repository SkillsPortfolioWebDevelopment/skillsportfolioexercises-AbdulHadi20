HTML & CSS REFERENCE GUIDE


HTML stands for HyperTextMarkupLanguage. It is used to build the skeletal structure of the websites. Every HTML tag is written inside <> brackets. Closing tags include a '/' with the name of the tag. Here is the reference guide for the basic tags:


The Tag Description Code Snippet
< html> This is the root element of the document. It converts the text documents into webpages. It also indicates the begining and the end of the document. 
 
                     
                        < html>
                        
                        < /html>
< head> This tag includes the meta information about the webpage. It tells the search engines what the website is all about. This helps with the SEOs. 
 
                     
                        < head>
                          < title>Title here< /title> 
                        < /head>
< title> The title tag displays its contents on the title bar of a web browser. It is essential to give accurate titles to websites/webpages for better results in SEOs and SERP. 
 
                     
                        < title>
                            Some Title
                        < /title>
< body> The body tag has all the content of the webpage/website inside it. It includes all the other tags inside it. 
 
                     
                        < body>
                            everything to be displayed....
                        < /body>
< h1> - < h6> The heading tags h1 to h6 are used for the headings and sub-headings in a webpage. The h1 is the largest of them all. Then onwards each heading gets smaller and smaller in size. h6 is the smallest. 
 
                     
                        < h1>Heading 1< /h1>
                        < h2>Heading 2< /h2>
                        < h3>Heading 3< /h3>
                        < h4>Heading 4< /h4>
                        < h5>Heading 5< /h5>   
                        < h6>Heading 6< /h6>
< p> The p tag is used to write paragraphs in a webpage. The paragraph's alignments can be done using CSS properties. 
 
                     
                        < p>
                          Lorem ipsum dolor sit amet consectetur adipisicing elit. 
                          Tempore provident consectetur facere dicta accusamus labore 
                          quaerat numquam id neque adipisci fugiat nisi, dolorem eaque 
                          eligendi cupiditate temporibus laboriosam nesciunt fugit. 
                        < /p>
< img> This tag is used to add images in a webpage. Images enhance the website quality and gives a better insight about the content. We also add the image address in quotes to locate the image and display it. 
 
                     
                        < img src="the aesthetic - rainy day.png" alt="an amazing photo">
< a> This is known as anchor tag. This is used for navigation or linking websites or a specific location on the current page. 
 
                     
                        < a href="https:www.google.com/" target="_blank">Visit Google now!!< /a>
< br> This is a line break element. It is used to force content into a new line. This tag is very essential as HTML does not know about whitespaces. 
 
                     
                        < br>
< hr> This tag is used to create horizontal lines in our webpages. Its length and thickness can be modified using CSS. 
 
                     
                        < hr>
< table> This tag is used to create tables. It is used at the begining and the end of the table. 
 
                     
                        < table>
                          < tr>
                           < th>some content< /th> 
                          < /tr> 
                        < /table>
< th>, < tr>, < td> th stands for table head. It is used to write headings of the table. tr stands for table rows. It is used to add content to table rows. td stands for table data. It is used to add content to table coloumns. 
 
                     
                        < tr>
                         < td>content< /td>
                        < /tr>



CSS stands for Cascading StyleSheet. It is used to modify/design the webpage. It is responsible for the visual representation of the webpages/websites.


CSS Properties Description Code Snippet
font- A short-form for all the font related properties. One of the most used properties in CSS. It can edit size, color, style, etc. of the text. 
 
                     
                        p {
                            font-size: large;
                            font-family: 'Courier New', Courier, monospace;
                            font-weight: bold;
                            font-style: italic;
                        }
background- A short-form for all the background related actions. It can perform actions for background color, image, etc. 
 
                     
                        p { 
                            background-color: aqua;
                            background-size: auto;
                        }
padding- A shorthand for the padding properties. Padding refers to the area around the certain element. 
 
                     
                        p {
                            padding-left: 12px;
                            padding-right: 12px;
                            padding-top: 12px;
                            padding-bottom: 12px;
                        }
border- A shorthand for border related modifications. It can modify the style, color, radius etc. of the borders. 
 
                     
                        p { 
                            border-collapse: collapse;
                            border-radius: 10px;
                            border-bottom: 12px;
                        }
align-content A property to align the items in a container when all the space is not being used. 
 
                     
                        p {
                            align-content: center;
                        }
float It specifies the position of elements wether the content will float to the left, right, center, etc. 
 
                     
                        p {
                            float: inline-end;
                        }
overflow The overflow property is used when it is required to specify the layout of the content in a container if the content is too big. 
 
                     
                        p {
                            overflow: auto;
                        }
text-decoration It is responsible for the modification of the text. It can make the text underlined, dotted, etc. 
 
                     
                        p {
                            text-decoration: dashed;
                        }
